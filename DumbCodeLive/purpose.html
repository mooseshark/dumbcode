What is Dumb Code?
What is Dumb Code?

Dumb Code is building something just for the fun of it. It doesn't need to be elaborate, or inventive. It doesn't need to change the world. It doesn't have to solve any existing problems. It doesn't even have to be good as long as you're having a good time doing it. If you set out to build anything, you're bound to learn something new. And if you learn something new, you've improved yourself. So, even if the code is dumb, you won't be dumb for writing the code.

Why is Dumb Code?
Why is Dumb Code?

Dumb Code exists because I write a lot of dumb things myself. Sometimes they lead to interesting new discoveries. Sometimes they turn into bad games. Sometimes I start with something that serves absolutely no purpose, only to learn the thing I made is actually useful in some way. Since I'm writing the code anyways, I thought I could add some notes and pictures and make blogs out of the weird projects I make. Maybe they'll help someone learn a concept, or inspire someone to build their own application, or even improve someone's outlook on programming when they're feeling a bit overwhelmed or stressed out by it.

When is Dumb Code?
When is Dumb Code?

Dumb Code is anytime you feel like it. You don't need a reason to make a dumb app, game, or web page. You just need some free time and an idea. So enjoy these blogs, get ideas, make something unique or pointless or stupid and have fun while you're doing it. You never know what you'll come up with!
